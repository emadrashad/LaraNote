Troubleshooting Guide

Common issues and solutions for LaraNote.

Toolbar Doesn't Appear on Text Selection

Possible Causes

  • The page was not fully loaded when you selected text.
  • The extension does not have the necessary permissions.
  • The website has a strict Content Security Policy (CSP) that blocks the toolbar.
  • You are on a protected browser page (e.g., `chrome://` pages).

Solutions

  • Refresh the page: Press Ctrl+R or Cmd+R to reload the page and try again.
  • Check Extension Status: Make sure the extension is enabled in your browser's extension settings. You might need to reinstall it if it's not working.
  • Try a different website: See if the issue persists on other sites. If it's site-specific, it might be a CSP issue.

My Highlights Disappeared

Possible Causes

  • You cleared your browser's cache or local storage.
  • The website's structure changed, and LaraNote can no longer find the highlighted text.
  • You are using a different browser profile.

Solutions

  • Check Browser History: Avoid clearing "Cookies and other site data" or "Cached images and files" for sites where you have notes.
  • Use Export/Import: Regularly export your notes for a specific page as a backup from the extension popup.
  • Dynamic Content: For highly dynamic websites (like social media feeds), highlights may be less reliable.

I Can't Export or Import Notes

Possible Causes

  • Browser is blocking pop-ups or file downloads.
  • The JSON file you are trying to import is corrupted or not in the correct format.

Solutions

  • Allow Downloads: Check if your browser is blocking file downloads from extensions. You may see an icon in the address bar.
  • Validate JSON: Ensure the file you are importing is a valid JSON file that was originally exported from LaraNote.
